#  Icon图标
用于常用的一些图标集合
***
采取的图标库v4.7.0版本[font-awesome](http://fontawesome.dashgame.com/)
## how to use
`<Icon type="hand-lizard-o"  color="red" size="50" spin/> `

## API 
| 属性  | 描述|  类型|  默认值|  
| ---------- | -----------| -----------|-----------|
| type   |   图标名称 | String|  - |
| size    | 图标大小(px)   |  Number or String  |   - |
| color   | 颜色    |   String |   - |
| spin   | 旋转动画   |   - |   - |
