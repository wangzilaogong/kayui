#  Button 按钮
用于常用的一些按钮
***

## how to use
`<Button type="primary"></Button> `
***


## API 
| 属性  | 描述|类型|基本值| 
| ---------- | -----------| -----------| -----------|
| type   |   按钮主题 ||default,primary,normal,warm,danger,disabled   |
| size   |   按钮大小 | 大(lg),默认,小型(sm),迷你(xs) | lg,sm,xs,-|
| long   |   按钮适应父容器 | | - |
| disabled   |   按钮状态 | |  |
| loading   |   按钮旋转 || true ,false |
| to   |   按钮跳转 | | 按钮主题 |
| target   |   a 标签的基本target值 |  |
